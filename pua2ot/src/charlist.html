Private Use to OpenType Reference Page

This page demonstrates all of the conversions that the Junicode utility pua2ot is capable of. If you want to know how to convert a Private Use character, either search for the hexadecimal code (e.g. E010) or for the character itself (you will find it in the “pua” column, which displays the character to be converted).

Next, check to make sure that any characters appearing in the last four columns match the Private Use character. If the last four columns are blank for the row you have located, the contents of the “base” column are the replacement. This must also match the Private Use character.

Click on a cell in the “base” column to see the Unicodes that make up the base. Click on any of the rightmost four columns to see the code that generates its content. The dialog that displays this information has a “Copy” button, which copies it to the clipboard.
