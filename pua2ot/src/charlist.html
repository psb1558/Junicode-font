Private Use to OpenType Reference Page

This page tests and demonstrates all of the conversions that the Junicode utility pua2ot is capable of. If you want to know how to convert a Private Use or other character, locate the row or rows of the character. Click on a cell in the “base” column to see the Unicodes that make up the base. Click on any of the rightmost five columns to see the code that generates its content. The dialog that displays this information has a “Copy” button, which copies it to the clipboard.

To filter the table, type the first digit or digits of a hexadecimal number (to search the “hex” column) or any other text (to search the “description” column) in the box below. For example, to find characters with hex codes beginning E0, type that sequence in the box; to find combining marks, type combining.
